# Product Positioning

## 1. Positioning Statement

一句话定位陈述（建议结构）：

> 对于 <目标用户>，<本产品> 是一个 <产品品类>，与 <主要替代方案> 相比，它的不同在于 <差异化关键>。

## 2. Target Market

- 主要市场（地区 / 行业 / 用户类型）
- 次要市场
- 暂不进入的市场

## 3. Core Value Proposition

- 用户价值（一条一句）
- 业务价值
- 不可被现有方案替代的关键点

## 6. Messaging Keywords

- 主关键词（3–5 个，可用于 PRD / 文案 / 设计 brief）
