## 维护规则

- 不写入真实 token、API key、secret。
- 不写入未脱敏的内部链接（例如包含 session id 的会话链接）。
- 公司内部资源（TAPD / 工蜂 / 腾讯文档 / KM 等）使用对外可识别的占位形式或脱敏后写入，不要包含敏感会话或 token 参数。
- 链接失效或被替换时，不要直接覆盖旧条目；新增一行并把旧条目标记为 `Deprecated`。
