    插入原型预览

    原型块仅保存在浏览器（localStorage），不会写入下载的 .md。可在评审时插入交互原型，评论可挂到块级。

    React / Vue 多文件项目就走这条：
    ① 本地：跑 npm run dev → 贴 http://localhost:xxxx
    ② 已部署：贴 GitHub Pages / Vercel / Netlify 的 URL
    ③ 沙箱：CodeSandbox / StackBlitz / CodePen 提供的 /embed/ URL
    Figma 编辑界面、Notion 文档等通过 X-Frame-Options 禁 iframe —— 那种情况点 iframe 角落的小标签会用新标签打开。

    代码会装进 iframe 的 srcdoc，受沙箱限制（不能访问父页 storage / DOM）。多文件 React/Vue 项目用上面 URL 模式更省事。
