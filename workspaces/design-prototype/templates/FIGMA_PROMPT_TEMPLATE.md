## 13. Final English Prompt

```text
Design a high-fidelity [platform] interface for [product/page].

Context:
- Target user: [target user]
- User scenario: [scenario]
- Product goal: [goal]

Visual style:
- Use [color direction] with [typography direction].
- Create a [density level] layout with clear hierarchy, generous whitespace, precise alignment, and polished component states.
- Use [radius], [shadow style], [icon style], and [motion tone].

Layout:
- Include [section 1], [section 2], [section 3].
- Prioritize [primary content] and make [primary action] visually clear.

Components:
- Include [component list].
- Provide default, hover, pressed, focused, disabled, loading, empty, error, and success states where relevant.

Responsive behavior:
- Optimize for [screen sizes].
- Preserve content hierarchy and touch usability on mobile.

Copywriting:
- Use concise, product-aware microcopy in [tone].

Constraints:
- Do not copy any existing product UI directly.
- Keep the design production-friendly and ready for engineering handoff.
```

## 14. Iteration Prompt

```text
Refine the design by improving [specific area]. Keep the same product goal and platform constraints. Make the visual hierarchy clearer, component states more complete, and mobile behavior more polished.
```
