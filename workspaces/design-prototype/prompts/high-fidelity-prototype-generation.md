# High-fidelity Prototype Generation Prompt

## 1. Prompt Name

High-fidelity Prototype Generation Prompt

## 2. Use Case

用于从产品想法或已澄清需求生成完整高保真交互原型方案。

## 3. Copyable Prompt

```text
请作为 AI Product Builder 的 Design Prototype Partner，根据下面的产品想法生成高保真交互原型方案。

默认中文输出。请先判断需求是否完整；如果缺少目标用户、场景、平台、产品目标、内容优先级或关键操作路径，请列为待确认问题，不要编造。

请输出：
1. 页面清单
2. 用户路径
3. 信息架构
4. 设计系统
5. 视觉风格
6. 页面布局
7. 组件说明
8. 交互说明
9. 空状态 / 加载状态 / 错误状态
10. 动效说明
11. Figma Prompt
12. 设计交付说明
13. 前端实现注意事项
14. 验收标准
15. 风险和待确认问题
16. 下一步行动建议

产品想法或需求：
[粘贴产品想法、PRD 或需求]
```

## 4. Expected Output

- 高保真原型方案
- 页面清单和用户路径
- 设计系统和视觉风格
- Figma Prompt
- 工程交付说明
