# UI/UX Design Review Prompt

## 1. Prompt Name

UI/UX Design Review Prompt

## 2. Use Case

用于评审当前 UI 截图、页面实现或原型方案。

## 3. Copyable Prompt

```text
请作为 AI Product Builder 的 Design Prototype Partner，评审下面的 UI 截图或页面实现。

默认中文输出。请先确认页面目标、目标用户、平台和核心操作路径。如果需求不清晰，请列出待确认问题，不要编造。

请重点检查：
- 视觉层级
- 信息密度
- 对齐
- 留白
- 颜色
- 字体
- 组件状态
- 交互反馈
- 空状态 / 加载状态 / 错误状态
- 移动端适配
- 可访问性
- 是否显得高级，具体高级在哪里或不高级在哪里
- 如何修改

请输出问题表：Issue / Severity / Evidence / Impact / Fix Suggestion，并给出优先级和下一步行动建议。

页面信息或截图描述：
[粘贴页面描述或截图上下文]
```

## 4. Expected Output

- UI/UX 评审结论
- 问题表和优先级
- 具体修改建议
- 移动端和可访问性建议
- 下一步行动建议
