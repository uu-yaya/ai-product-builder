# Figma Prompt Generation Prompt

## 1. Prompt Name

Figma Prompt Generation Prompt

## 2. Use Case

用于把产品需求转化为 Figma Make、Figma AI、v0 或 Cursor 可用的英文设计 Prompt。

## 3. Copyable Prompt

```text
请作为 AI Product Builder 的 Design Prototype Partner，把下面的产品需求转化为可用于 Figma Make / Figma AI / v0 的英文设计 Prompt。

默认先用中文做设计拆解，再输出英文 Prompt。请先判断需求是否完整；如果缺少目标用户、使用场景、平台、视觉定位、页面清单、内容优先级或关键操作路径，请列为待确认问题，不要编造。

请输出：
1. 中文设计拆解
2. 页面级说明
3. 组件级说明
4. 交互状态
5. 空状态 / 加载状态 / 错误状态
6. 动效说明
7. 视觉 token：颜色、字体、间距、圆角、阴影、图标风格
8. 英文 Prompt
9. 可迭代 Prompt
10. 下一步行动建议

产品需求：
[粘贴需求]
```

## 4. Expected Output

- 中文设计拆解
- 英文 Figma Prompt
- 页面级和组件级说明
- 状态、动效、视觉 token
- 可迭代 Prompt
