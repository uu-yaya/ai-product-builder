# Design Inspiration Research Prompt

## 1. Prompt Name

Design Inspiration Research Prompt

## 2. Use Case

用于根据产品目标和用户场景做设计灵感研究，并转化为当前产品可用的设计方向。

## 3. Copyable Prompt

```text
请作为 AI Product Builder 的 Design Prototype Partner，帮我为下面的产品做设计灵感研究。

默认中文输出。请先根据产品目标、用户场景、平台和视觉定位生成搜索关键词。优先参考 Figma Community、Dribbble、Mobbin、Awwwards、Behance、Pinterest、Apple HIG、Material Design 等来源；如果涉及最新设计趋势、竞品界面或具体工具能力，请联网验证并标注来源。

请不要照抄具体作品，只提炼视觉风格、交互模式、信息架构、组件模式和可适配原则。

请输出：
1. 设计研究目标
2. 搜索关键词
3. 推荐参考来源及适合参考的内容
4. 参考分析表：Reference / Visual Style / Interaction Pattern / What to Learn / What Not to Copy / How to Adapt
5. 可用于当前产品的设计方向
6. 风险和待确认问题
7. 可用于 Figma 的方向性英文 Prompt
8. 下一步行动建议

产品信息：
[粘贴产品目标、用户场景、平台和约束]
```

## 4. Expected Output

- 搜索关键词
- 参考来源和分析表
- 不照抄的设计原则
- 当前产品的设计方向
- Figma 方向性英文 Prompt
