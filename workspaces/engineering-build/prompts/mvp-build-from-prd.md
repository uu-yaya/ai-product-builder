# MVP Build from PRD Prompt

## 1. Prompt Name

MVP Build from PRD Prompt

## 2. Use Case

用于把 PRD / 原型 / 设计交付转成 MVP 落地计划。

## 3. Copyable Prompt

```text
请作为 AI Product Builder 的 Engineering Build Partner，把下面的 PRD / 原型 / 设计交付转成 MVP 落地计划。

默认中文输出。请先提取 MVP 范围和非目标，不要直接写代码，先输出实现计划。

请拆分以下任务：
- Frontend
- Backend
- API
- Data Model
- AI / Algorithm
- QA
- DevOps / Launch

请输出依赖关系、优先级、验收标准、风险和测试方式。

请输出：
1. Idea / Requirement Summary
2. User Problem
3. MVP Scope
4. Non-goals
5. User Flow
6. Feature List
7. Tech Stack
8. Frontend Plan
9. Backend Plan
10. Data Model
11. API Plan
12. AI Workflow
13. Testing Plan
14. Launch Plan
15. Risks
16. Timeline
17. Acceptance Criteria
18. 下一步建议

输入内容：
[粘贴 PRD / 原型 / 设计交付]
```

## 4. Expected Output

- MVP 落地计划
- 分模块任务表
- 优先级和依赖关系
- 验收标准
- 风险和测试方式
