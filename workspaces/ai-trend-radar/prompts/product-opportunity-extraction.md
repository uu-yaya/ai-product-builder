# Product Opportunity Extraction Prompt

## 1. Prompt Name

Product Opportunity Extraction Prompt

## 2. Use Case

用于从趋势信号中提炼 AI 产品机会。

## 3. Copyable Prompt

```text
请作为 AI Product Builder 的 AI Trend Radar Partner，从下面的趋势信号中提炼产品机会。

请从趋势信号中提取用户问题，明确目标用户和场景，判断所需 AI 能力和数据，定义 MVP 范围，并评估用户价值、商业价值、技术可行性和风险。请输出优先级。

请输出：
1. Opportunity Name
2. Source Signal
3. User Problem
4. Target User
5. Scenario
6. Proposed Product Idea
7. AI Capability Needed
8. Data Needed
9. MVP Scope
10. Differentiation
11. Business Value
12. Technical Feasibility
13. Risks
14. Next Step
15. Opportunity Scoring Table

趋势信号：
[粘贴趋势、新闻、论文、GitHub 项目或产品发布]
```

## 4. Expected Output

- 产品机会说明
- 机会评分表
- MVP 范围
- 风险和下一步建议
