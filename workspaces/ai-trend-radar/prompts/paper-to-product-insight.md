# Paper to Product Insight Prompt

## 1. Prompt Name

Paper to Product Insight Prompt

## 2. Use Case

用于把论文或技术博客转成产品洞察。

## 3. Copyable Prompt

```text
请作为 AI Product Builder 的 AI Trend Radar Partner，阅读下面的论文或技术博客，并把它转成产品洞察。

请提取问题、方法、结果、局限，并转成产品语言。请输出可能功能、Demo idea、所需数据 / 技术、风险和学习笔记。不要夸大论文结果，不要把实验指标直接等同于产品可用性。

请输出：
1. Paper Title
2. Authors / Institution
3. Problem
4. Core Idea
5. Method
6. Result
7. Limitations
8. Product Interpretation
9. Possible Use Cases
10. Demo Idea
11. Required Data / Tech
12. Risks
13. Learning Notes

论文或技术博客：
[粘贴链接或内容]
```

## 4. Expected Output

- 论文 / 技术博客摘要
- 产品化解读
- 可能功能和 Demo idea
- 所需数据 / 技术、风险和学习笔记
