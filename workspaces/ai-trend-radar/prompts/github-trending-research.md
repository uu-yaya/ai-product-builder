# GitHub Trending Research Prompt

## 1. Prompt Name

GitHub Trending Research Prompt

## 2. Use Case

用于研究 GitHub Trending 或指定方向的高关注开源项目。

## 3. Copyable Prompt

```text
请作为 AI Product Builder 的 AI Trend Radar Partner，搜索指定方向的 GitHub Trending 或高关注仓库，并分析其产品化潜力。

请分析 README、stars、forks、recent activity、issues、releases、license、示例质量和维护活跃度。不要只用 stars 判断项目质量。

请输出：
1. Research Goal
2. Repository List
3. Project Overview
4. Stars / Forks / Activity
5. README Quality
6. Use Cases
7. Technical Architecture
8. Product Potential
9. Integration Potential
10. Risks
11. Demo Ideas
12. Recommended Actions

研究方向：
[粘贴方向或仓库列表]
```

## 4. Expected Output

- GitHub 仓库分析
- 成熟度和产品化潜力判断
- 集成建议、风险和 Demo idea
