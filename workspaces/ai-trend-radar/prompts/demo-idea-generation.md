# Demo Idea Generation Prompt

## 1. Prompt Name

Demo Idea Generation Prompt

## 2. Use Case

用于基于趋势、论文、GitHub 项目或产品发布生成可落地 Demo idea。

## 3. Copyable Prompt

```text
请作为 AI Product Builder 的 AI Trend Radar Partner，基于下面的趋势、论文、GitHub 项目或产品发布生成一个可落地 Demo idea。

请明确用户问题、核心体验、AI 能力、数据 / API、关键页面、MVP Flow、技术栈建议、成功标准、风险和构建计划。

如果适合落地，请建议同步到 `workspaces/engineering-build/`。
如果需要产品定义，请建议同步到 `workspaces/pm-strategy/`。
如果需要原型，请建议同步到 `workspaces/design-prototype/`。

请输出：
1. Demo Name
2. Inspiration Source
3. Problem
4. Target User
5. Core Experience
6. AI Capability
7. Data / API Needed
8. Key Screens
9. MVP Flow
10. Tech Stack Suggestion
11. Success Criteria
12. Risks
13. Build Plan
14. Workspace Sync Recommendation

灵感来源：
[粘贴趋势、论文、GitHub 项目或产品发布]
```

## 4. Expected Output

- 可落地 Demo idea
- MVP Flow 和关键页面
- 技术栈建议
- 风险、成功标准和构建计划
- 工作区同步建议
