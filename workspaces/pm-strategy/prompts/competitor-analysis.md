# Competitor Analysis Prompt

## 1. Prompt Name

Competitor Analysis Prompt

## 2. Use Case

用于分析竞品，提炼差异化机会和产品行动。

## 3. Copyable Prompt

```text
请作为 AI Product Builder 的 PM Strategy Partner，分析下面的竞品或产品类别。

默认中文输出。请先判断输入信息是否完整；缺失信息列为待确认问题，不要编造。

涉及最新竞品、市场、AI 技术、价格、产品发布或外部资料时，必须联网验证并标注信息来源。

请输出结构化内容，至少包含：
1. 分析目标
2. 竞品选择标准
3. 信息来源
4. Competitor Overview
5. Target Users
6. Core Use Cases
7. Feature Map
8. AI Capabilities
9. User Journey
10. UX Highlights
11. Pricing / Business Model
12. Strengths
13. Weaknesses
14. Learnings
15. 不应照抄的点
16. Differentiation Opportunities
17. Recommended Actions
18. 下一步行动建议

分析对象：
[粘贴竞品名称、链接或产品类别]
```

## 4. Expected Output

- 竞品对比表
- 信息来源
- 差异化机会
- 不应照抄的点
- 推荐行动
