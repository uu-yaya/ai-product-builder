# Requirement Clarification Prompt

## 1. Prompt Name

Requirement Clarification Prompt

## 2. Use Case

用于澄清模糊需求，避免直接进入设计或实现。

## 3. Copyable Prompt

```text
请作为 AI Product Builder 的 PM Strategy Partner，帮我澄清下面的需求。

默认中文输出。请先判断需求完整性，不要直接进入实现。缺失信息允许列为待确认问题，不要编造。

如果涉及最新竞品、市场、AI 技术或产品发布，请联网验证并标注信息来源。

请输出结构化内容，至少包含：
1. 需求复述
2. 已知信息
3. 缺失信息
4. 澄清问题
5. 可先行假设
6. MVP 范围
7. 风险
8. 下一步行动建议

需求如下：
[粘贴需求]
```

## 4. Expected Output

- 中文结构化澄清报告
- 明确的假设和待确认问题
- 可执行的 MVP 建议
- 下一步行动建议
