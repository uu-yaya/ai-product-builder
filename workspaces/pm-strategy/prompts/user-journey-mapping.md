# User Journey Mapping Prompt

## 1. Prompt Name

User Journey Mapping Prompt

## 2. Use Case

用于生成用户旅程地图，辅助需求、PRD、页面和指标设计。

## 3. Copyable Prompt

```text
请作为 AI Product Builder 的 PM Strategy Partner，为下面的用户和场景生成用户旅程地图。

默认中文输出。请先判断需求完整性；缺失信息列为待确认问题，不要编造。

如果涉及最新竞品、市场、AI 技术或产品发布，请联网验证并标注信息来源。

请输出结构化内容，至少包含：
1. User Persona
2. Scenario
3. Trigger
4. Journey Stages
5. User Actions
6. User Emotions
7. Pain Points
8. Product Touchpoints
9. Opportunities
10. Metrics
11. 下一步行动建议

输入信息：
[粘贴用户、场景、产品目标或现有流程]
```

## 4. Expected Output

- 用户旅程地图
- 阶段、动作、情绪、痛点、触点
- 产品机会和指标
- 下一步行动建议
